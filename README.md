Procedure to Run the Program
Open Command Prompt or Terminal.
Navigate to the directory where the Java files are saved.
Compile the calculator program using the command javac Calculator.java.
Run the calculator program using the command java Calculator.
Enter the choice of arithmetic operation.
Enter the two numbers as input.
Observe the result displayed on the screen.
Compile the API program using the command javac CalculatorAPI.java.
Run the API program using the command java CalculatorAPI.
Ensure the server starts successfully at port 8000.
Open a web browser.
Enter the API URL with parameters like http://localhost:8000/calculate?a=10&b=5&op=add.
Check the result returned by the API in the browser.
Test all arithmetic operations using different URLs.
Verify the correctness of the outputs.
